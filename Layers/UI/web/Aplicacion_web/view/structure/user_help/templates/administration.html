#HELP_ADMINISTRATION_1#:

Notificaciones

Desde aquí, podemos ver un listado de notificaciones realizadas por los usuarios del Sistema avisando sobre errores o bien sugiriendo nueva funcionalidad. También es posible marcar una notificación como resuelta o eliminarla.

Actividades

Muestra todas las actividades del sistema, permitiendo modificar las actuales o crear nuevas. Así mismo, permite añadir o quitar profesores de una actividad. O asociar una actividad a una competencia

Competencias

Muestra todas las competencias existentes en el sistema. Permite modificarlas, añadir nuevas competencias y asignar indicadores a competencias.

Usuarios

Muestra todos los usuarios del sistema junto a sus datos personales. Permite que estos sean modificados.
